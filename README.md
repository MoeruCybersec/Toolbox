## 安装

### 1. 克隆项目到本地

```bash
$ mkdir -p "$HOME/hacking"
$ git clone "https://github.com/MoeruCybersec/Toolbox.git" "$HOME/hacking/toolbox"
```

### 2. 添加环境变量 (可选)

将以下内容添加到你的 shell 配置文件 (例如 `.zshrc`, `.bashrc`)

```bash
export HKTB="$HOME/hacking/toolbox"
export PATH="$HKTB/link:$PATH"
```
